# Styling in CSS

CSS is a cornerstone technology for styling web applications. Here's how you can style a simple button:

```css
button {
  background-color: #4CAF50;
  color: white;
  padding: 15px 32px;
  text-align: center;
  font-size: 16px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

button:hover {
  background-color: #45a049;
}
```

This code defines a green button with hover effects.
